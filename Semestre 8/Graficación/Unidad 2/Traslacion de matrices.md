# Rotacion

Rotacion bidimiencional sobre el eje z
\[Qx\]     \[Cos(θ) -(Sin(θ)) 0\] \[Px\] 
\[Qy\] = \[Sin(θ)   Cos(θ)   0\] \[Py\] 
\[1\]       \[0            0         1\]  \[1\] 

Rotacion tridimencional sobre el eje z
\[Qx\]     \[1          0            0        0\] \[Px\] 
\[Qy\] = \[Cos(θ) -(Sin(θ))  0        0\] \[Py\] 
\[Qz\]     \[Sin(θ)   Cos(θ)   0        0\]  \[Pz\] 
\[1\]       \[0          0            0        1\]  \[1\]
